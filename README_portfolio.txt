# App Mudanza

App Mudanza is an Android app designed to support the moving process with a simple, practical mobile experience.

## What I built
- Android app developed with Kotlin and Jetpack Compose.
- Release build configured with a signed keystore.
- Beta distribution set up through Firebase App Distribution.
- Public download option available through a GitHub Release APK.

## Release workflow
1. Build the app in release mode.
2. Sign the APK with a release keystore.
3. Distribute internal beta builds through Firebase App Distribution.
4. Share a public APK download link through GitHub Releases for easy demo access.
